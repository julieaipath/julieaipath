### 三、 Vibe Coding 提效实战：怎么让 AI 听懂人话？

配置好工具只是第一步，**怎么提需求**才是拉开效率差距的关键。

- **简单粗暴型**：如果你只是想改个按钮文字，直接说：“把 index\.html 里的‘提交’改成‘确认’。”它秒级响应，直接到位。
  
- **逻辑拆解型（避坑重点）**：新手最怕prompt指令模糊。正确的 Vibe coding 应该是：“找到 id 为 clearBg 的清除按钮，把它的背景改成红色，记得在 CSS 文件里同步改掉。” 这种清晰的指令，AI 会自动去翻你的文件，把那些散落在各处的代码逐一修正，这才是真正的降维打击。
